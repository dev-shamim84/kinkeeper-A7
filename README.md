# KeenKeeper — Keep Your Friendships Alive

## 📖 Description
KeenKeeper is a modern web application designed to help users maintain and strengthen their friendships. In today’s busy life, it is easy to lose touch with people we care about. This app solves that problem by allowing users to track their interactions with friends, set communication goals, and get reminders when it’s time to reconnect.

Users can view their friends, check how long it has been since the last interaction, and quickly perform actions like calling, texting, or video chatting. Every interaction is recorded in a timeline, helping users build consistent and meaningful relationships over time.

## 🛠️ Technologies Used
* React.js
* Next.js
* JavaScript (ES6+)
* Tailwind CSS
* Recharts
* Npm Package 
* React-hot-toast
* JSON (for data handling)

## ✨ Key Features

1. **Friend Tracking System**
   Easily manage and monitor your friends with detailed profiles, including contact history, tags, and relationship goals.

2. **Interactive Timeline**
   Automatically logs all interactions (Call, Text, Video) with date and activity type, helping users keep a clear history of communication.

3. **Analytics Dashboard**
   Visual representation of interaction data using charts, allowing users to understand their communication patterns at a glance.
